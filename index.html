Introduction

Python is an interpreted, high level, general purpose and object-oriented or structured programming language. It was created by Guido van Rossum in 1991 and developed by Python Software Foundation.

It was especially developed to emphasize code readability, and its syntax allows programmers to express logic and instructions in fewer lines of code. Therefore, we can state that Python is a very synthetic programming language, which allows us to store and to manipulate a large amount of data easier.

The main features of Python:

  • Simple and easy to learn
  • Free and Open Source
  • Cross Plattform
  • High-level Interpreted Language
  • Object Oriented

    Hello World

    Open your Python interpreter or code editor and type the following in order to output your first "Hello World":

    >>> print "Hello World" Hello World
    Variables and Data Types

    A value is one of the most important things that a software can manipulate. Python has two different data types: number (9, e.g.), which is an integer, and string ("Hello World", e.g.)

    >>> print 9 9 >>> print "Hello World" Hello World
    Assignment Statement

    One of the greatest features of a programming language is the power to manipulate variables. An assignment statement creates new variables and provides them values.

    >>> message = "My name is Bond, James Bond" >>> n = 17 >>> pi = 3.14159

    The print statement also works with variables:

    >>> print message My name is Bond, James Bond >>> print n 17 >>> print pi 3.14159
    Lists

    Lists are among the most used data types in Python. Declaring a list is very simple. You just need to insert all the items within brackets [ ] and separate them by commas.

    >>> a = [1, 2.2, 'python']
    If...else Statements

    If statement starts with a condition that produces a result if the condition is satisfied. Else produces another result if the condition is not satisfied. In the example below, a simple Python program verifies if a number is positive using if...else statements:

    if num >= 0: print("Positive or Zero") else: print("Negative number")
